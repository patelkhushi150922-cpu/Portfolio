## ⚙️ Deploy to GitHub Pages (Free Hosting)

Follow these steps to get your portfolio live at `https://patelkhushi150922-cpu.github.io/Portfolio/`:

### Step 1 — Upload the file
1. Go to your repo: [github.com/patelkhushi150922-cpu/Portfolio](https://github.com/patelkhushi150922-cpu/Portfolio)
2. Click **"creating a new file"** (or drag and drop `index.html`)
3. Name it `index.html`
4. Paste the full HTML code
5. Click **"Commit changes"**

### Step 2 — Enable GitHub Pages
1. Go to your repo **Settings** tab
2. In the left sidebar, click **Pages**
3. Under "Source", select **Deploy from a branch**
4. Choose branch: `main`, folder: `/ (root)`
5. Click **Save**

### Step 3 — Wait 1–2 minutes
Your site will be live at:
```
https://patelkhushi150922-cpu.github.io/Portfolio/
```

## ✏️ How to Update Projects

Open `index.html` and find the `<!-- PROJECTS -->` section. Each card looks like this:

```html
<div class="project-card" data-type="wordpress">
  <div class="project-header">
    <div class="project-title">Project Name</div>
    <span class="project-tag">WordPress</span>
  </div>
  <div class="project-desc">Short description of the project.</div>
  <a class="project-link" href="https://yoursite.com/" target="_blank">Visit Site ↗</a>
</div>
```

Change `data-type` to `wordpress`, `shopify`, or `ai` for filtering to work correctly.
